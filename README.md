# CaterToUs

## Project Description 

CaterToUs is an AI-powered catering resource management platform designed to help catering businesses reduce food waste, optimize inventory usage, and improve operational efficiency.

The platform provides catering companies with real-time inventory tracking, expiry monitoring, portion forecasting, supplier management, and waste analytics. By leveraging predictive insights and smart resource planning, CaterToUs helps businesses make data-driven decisions that contribute to a more sustainable future.

Built for the DoubleDot Smarter Resource Management Challenge at ImagineHack 2026.

## Challenge Statement

Businesses often struggle with inefficient resource allocation, resulting in unnecessary costs, excess inventory, and significant food waste.

For catering companies, inaccurate demand forecasting can lead to:

* Over-ordering ingredients
* Excess prepared food
* Increased disposal costs
* Reduced sustainability performance

These inefficiencies impact profitability while contributing to environmental issues such as food waste and carbon emissions.

---

## Our Approach

CaterToUs addresses these challenges through five core modules:

### Inventory & Expiry Tracking

Monitor ingredient quantities and expiry dates in real time to prevent spoilage and improve stock utilization.

### Smart Menu Planning

Track recipes and ingredient consumption to ensure available inventory is used efficiently before expiration.

### Portion Predictor

Estimate food requirements for upcoming events based on attendee numbers and event details, helping prevent overproduction.

### Supplier Order Management

Provide visibility into supplier orders and inventory requirements to streamline procurement decisions.

### Waste & Impact Analytics

Record food waste and visualize environmental impact, enabling businesses to identify patterns and improve sustainability practices.

## Installation & Setup

### Prerequisites

* Node.js (v18 or later)
* npm

### Clone the Repository

```bash
git clone <repository-url>
cd ImagineHack2026_WIMDF
```

### Backend Setup

```bash
cd server
npm install
```

Create a `.env` file:

```env
GROQ_API_KEY=your_key_here
TELEGRAM_BOT_TOKEN=your_token_here
USDA_API_KEY=your_key_here
```

Start the backend server:

```bash
npm run dev
```

or

```bash
npm start
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

## Usage

1. Add or update inventory items.
2. Monitor ingredients nearing expiration.
3. Create and manage recipes.
4. Forecast food portions for upcoming events.
5. Review supplier requirements.
6. Log food waste and monitor sustainability impact.
